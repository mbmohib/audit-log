### Install & Run

1.  First clone the repo:

```bash
$ git clone git@github.com:mbmohib/audit-log.git
```

2.  Then go to the folder:

```bash
$ cd audit-log
```

3.  Then install the dependencies:

```bash
$ npm install
```

4.  Finally run the app:

```bash
$ npm start
```

5.  For Running test:

```bash
$ npm run test
```
